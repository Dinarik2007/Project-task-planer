# Планировщик задач (лендинг + приложение)

Небольшой фронтенд-проект на чистом HTML/CSS/JavaScript: приветственный лендинг и полноценный планировщик задач для маркетинговой команды в одном файле.

## Возможности

- Приветственный лендинг с быстрым переходом к рабочей области
- Роли пользователей (директор и сотрудники) с разграничением прав
- Создание задач, изменение статуса, комментарии и история действий
- Фильтры, поиск и визуальная индикация по дедлайнам
- Экспорт в CSV/JSON и импорт из JSON
- Локальное хранение данных в `localStorage`

## Структура проекта

- `index.html` — точка входа репозитория
- `планировщик.html` — основная страница лендинга и планировщика
- `.gitignore` — исключения для Git

## Быстрый старт

1. Склонировать репозиторий.
2. Открыть `index.html` в браузере.
3. Работать с задачами прямо в интерфейсе (без сборки и зависимостей).

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
